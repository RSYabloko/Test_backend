## Запуск

- Создать venv в корневой папке проекта: `python -m venv venv`
- Установить зависимости: `pip install -r requirements.txt`
- В папке backend переименовать `.env.example` -> `.env` и настроить его:
    - Если используется PostgreSQL:
        - Создать пустую БД
        - Добавить данные авторизации БД в `.env`
        - Открыть файл `.env` и расскоментировать строку с выбором БД (5 строка) и оставить только свою БД
        - Сделать миграцию: `python manage.py migrate`
        - Применить миграцию: `python manage.py makemigrations`
    - Если используется SQLite:
        - Сделать миграцию: `python manage.py migrate`
        - Применить миграцию: `python manage.py makemigrations`
- Создание суперпользователя:
    - В корневой папке backend выполняем `python manage.py createsuperuser`
    - Вводим Email и пароль, которые будут использоваться для авторизации
- Запуск сервера: `python manage.py runserver 0.0.0.0:8000`
- В браузуре заходим по адресу на котором запустили
